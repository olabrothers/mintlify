Sip is a design tool that allows you to collect, organize and edit colors. It also gives you the ability to share colors with everyone and link them between the other tools in your workflow.

Sip is designed to be used quickly, to be as present and invisible as possible. It's meant to be there every-time you need it and to go away before you know it.

Sip was created by Olá brothers, and it’s available for Mac.

## **Minimum Requirements**

To use the latest version of Sip, you must be running macOS Sonoma or newer.
